# Customer Sales Data Analysis using Python

## Project Overview
This project performs professional sales data analysis using Pandas.
It demonstrates data cleaning, filtering, grouping, visualization,
and generation of business insights.

---

## Features

- Load CSV dataset using Pandas
- Clean missing and duplicate data
- Generate summary statistics
- Perform filtering and grouping
- Create graphs using Matplotlib
- Generate meaningful insights
- Professional Python coding structure

---

## Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib
